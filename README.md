# Power BI Project Management Dashboard

This repository contains a **Power BI dashboard** built to analyze and monitor project performance for a fictional organization. It provides executives, project managers, and analysts with actionable insights on project KPIs, budgets, workload distribution, status, priority, and geographic performance.
